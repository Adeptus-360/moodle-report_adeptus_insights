Third-party libraries in this folder
=====================================

Lottie Web
----------
Download URL: https://github.com/airbnb/lottie-web
Version: 5.12.2
License: MIT
Files: lottie.js, lottie.min.js

To update:
1. Download the latest release from https://github.com/airbnb/lottie-web/releases
2. Replace lottie.js and lottie.min.js with the new versions
3. Update version number in thirdpartylibs.xml
