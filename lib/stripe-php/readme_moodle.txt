Stripe PHP Library
==================

This directory contains the Stripe PHP library for payment processing.

Download URL: https://github.com/stripe/stripe-php
Version: 16.3.0
License: MIT

The library is used by this plugin to handle Stripe payment processing
for subscription management.

Changes made:
- None. This is the unmodified library from the official repository.

To update:
1. Download the new version from https://github.com/stripe/stripe-php/releases
2. Extract and replace the contents of this directory
3. Update the version number in thirdpartylibs.xml
4. Test thoroughly before deploying
