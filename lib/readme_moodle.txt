Third-party libraries in this folder
=====================================

Chart.js
--------
Download URL: https://www.chartjs.org/
Version: 4.4.1
License: MIT
Files: chart.umd.min.js

To update:
1. Download the latest UMD build from https://cdnjs.com/libraries/Chart.js
2. Replace chart.umd.min.js with the new version
3. Update version number in thirdpartylibs.xml
